=== Automacao de calculos na Calculadora do iOS ===

0.Objetivo
 .Realizar calculos basicos na calculadora do iOS no smartphone

1.Requisitos:
 .Os drivers devem estar na pasta drivers
 .Java JDK 8
 .Intellij Idea ou Eclipse IDE
 .Dependencias Maven (Ja estao inseridas no arquivo pom.xml pronto para serem baixadas ao importar na IDE
  com as versoes estaveis).

2.Configuracoes:
 .O projeto foi criado no MacOS, não e para ser executado no Windows e no Linux. Automação com iOS so podem ser feitas na plataforma com o MacOS.
 .Os metodos para criar o driver Mobile estao na classe EmuladorDriver e MobileDriver

3.Execucao:
 .Abrir uma das classes ***Test.java no Intellij Idea ou no Eclipse
 .No caso para executar num smartphone, abra a classe Mobile Driver e altere os valores das variáveis

4.Evidencias:
 .Apos a execucao as imagens de evidencias sao armazenadas na pasta screenshots.

5.Massas:
 .Nao e necessario criar massas por enquanto.

6.App
 .O app e de terceiro, provavelmente você notara alguns problemas com identificadores de elementos, caso execute com sucesso ou não, recomendo que após entender o 
código escrito, instale outro arquivo.ipa. 
 .Antes de executar, pesquise instruções sobre o Apple ID e WebDriverAgent.
